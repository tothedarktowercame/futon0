# Excursion: Frames in WebArxana

**Date:** 2026-04-13
**Entry point:** M-daily-scan frame schema → "these EDN files should be viewable"

Daily scan frames (`futon5a/data/frames/*.edn`) should be viewable in
WebArxana — each frame as a diagram with leads, instruments, and cardinal
direction as nodes. Closes the loop: daily scan → frame → viewable in the
tool built for this kind of structure. Also demonstrates WebArxana to
potential users.
